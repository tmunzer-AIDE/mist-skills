# Wi-Fi Standard & Capability Reference

This data is NOT available from any MCP tool. Use these tables to classify devices.

## AP Models by Wi-Fi Standard

| Standard | Models |
|---|---|
| Wi-Fi 7 (802.11be) | AP36, AP36M, AP37, AP47, AP47D, AP47E, AP66, AP66D, AP723H |
| Wi-Fi 6E (6GHz capable) | AP33, AP34, AP45, AP45E, AP47, AP47D, AP47E, AP36, AP36M, AP37, AP63, AP63E, AP64, AP66, AP66D, AP723H |
| Wi-Fi 6 (802.11ax, no 6GHz) | AP32, AP32E, AP43, AP43E, AP43-FIPS, AP43E-FIPS, AP61, AP61E |
| Wi-Fi 5 (802.11ac) | AP41, AP41E, AP21, AP12 |
| Legacy (Wi-Fi 4) | AP17, AP24, AP27, AP27E |

## Special Capabilities

| Capability | Models |
|---|---|
| GPS (outdoor) | AP63, AP63E, AP64, AP66, AP66D |
| Wi-Fi 7 + 6GHz | AP36, AP36M, AP37, AP47, AP47D, AP47E, AP66, AP66D, AP723H |

## Notes

- All Wi-Fi 7 APs also support 6GHz (Wi-Fi 7 is a superset of Wi-Fi 6E)
- GPS models are outdoor APs only
- Mist inventory contains only Juniper/Mist devices. Cisco, Aruba, etc. will never appear.
- Mist-compatible switch families: EX2300, EX3400, EX4000, EX4100, EX4300, EX4400, EX4600, EX4650, EX5200, QFX series
- NOT Mist-compatible: EX2200 (too old)
